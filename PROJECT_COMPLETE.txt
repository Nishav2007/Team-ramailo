   HOW LIVE UPDATES WORK
================================================================

DASHBOARD AUTO-REFRESH:
   <meta http-equiv="refresh" content="30">
   
   This meta tag tells browser:
   - Reload page every 30 seconds
   - Fetch fresh data from database
   - Update status display
   - No JavaScript needed!

ADMIN PANEL AUTO-REFRESH:
   Same meta tag, 30 second refresh
   - Reloads location cards
   - Updates statistics
   - Shows latest status

HISTORY PAGE AUTO-REFRESH:
   60 second refresh (slower for less server load)
   - Updates events table
   - Shows new events

COUNTDOWN TIMER:

   "(30s) Dashboard" ‚Üí "(29s)" ‚Üí "(28s)" ... "(1s)" ‚Üí Refresh!
